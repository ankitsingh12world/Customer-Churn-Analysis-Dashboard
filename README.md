# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing customer churn using Power BI.
The goal is to identify key factors that influence customer churn and provide actionable insights to improve customer retention.

---

## 🎯 Objectives

* Analyze customer churn behavior
* Identify key drivers of churn
* Build an interactive dashboard for business insights
* Enable filtering and dynamic analysis using slicers

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning)
* DAX (Measures & KPIs)

---

## 📂 Dataset Features

The dataset includes the following attributes:

* CustomerID
* Gender
* Age
* Tenure
* Contract_Length
* Subscription_Type
* Total_Spend
* Support_Calls
* Usage_Frequency
* Payment_Delay
* Churn (0 = No, 1 = Yes)

---

## 📊 Dashboard Features

### 🔹 KPIs

* Total Customers
* Churn Customers
* Churn Rate (%)

### 🔹 Visualizations

* Churn by Contract Type
* Churn by Gender
* Churn by Subscription Type

### 🔹 Filters (Slicers)

* Gender
* Contract Length
* Subscription Type

---

## 📈 Key Insights

* Customers with **monthly contracts** show higher churn compared to long-term contracts.
* **Gender is not a strong factor** influencing churn.
* Customers with **basic subscription plans** have higher churn rates, indicating lower engagement or satisfaction.

---

## 🧠 Business Recommendations

* Encourage customers to shift to long-term contracts.
* Improve services or engagement for basic plan users.
* Focus on retention strategies for high-risk segments.
